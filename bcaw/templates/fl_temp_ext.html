BitCurator Access Disk Image Browser (bca-webtools)

{#
 
#}
 

This software enables access to file systems contained within forensically-packaged and raw disk images using a web browser. Supported file systems include FAT16, FAT32, NTFS, HFS+, ISO9660, and EXT2/3/4. Click on the 'Browse' icon to navigate through the file system(s) within the disk image, or the 'Download' icon to download the complete disk image.
